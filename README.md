# Ejecutar en desarrollo

1. Clone repository
2. Execute
```
yarn install
```
3. Install Nest CLI 

```
npm i -g @nestjs/cli
```

4. Run the database
```
docker-compose up -d
```

## Stack
* MongoDB
* Nest
